<feature>
  <name>Vacation Period Grouping Algorithm</name>
  <files>lib/vacation/types.ts, lib/vacation/periods.ts, lib/vacation/periods.test.ts</files>
  <behavior>
    groupVacationPeriods(vacationDates, holidayDates, year) groups vacation dates into periods.

    Rules:
    - Filter vacationDates to requested year (startsWith `${year}-`)
    - Sort chronologically
    - Two vacation days are in the same period if every day between them is either: a weekend day (Sat/Sun), a holiday, or another vacation day
    - A working day gap (weekday that is not vacation or holiday) breaks into separate periods
    - Return periods in reverse chronological order (most recent first)
    - Each period has: startDate, endDate, days[], dayCount

    Test cases:
    - Empty input -> []
    - Single day ["2026-03-02"] -> [{ startDate: "2026-03-02", endDate: "2026-03-02", days: ["2026-03-02"], dayCount: 1 }]
    - Mon-Fri consecutive ["2026-03-02", "2026-03-03", "2026-03-04", "2026-03-05", "2026-03-06"] -> 1 period with dayCount 5
    - Fri + Mon (weekend bridge) ["2026-03-06", "2026-03-09"] -> 1 period with dayCount 2 (Sat/Sun are weekend)
    - Mon + Wed with Tue as working day -> 2 separate periods (working day gap)
    - Mon + Wed with Tue as holiday -> 1 period (holiday bridges)
    - Mon-Tue + Thu-Fri with Wed as holiday -> 1 period (4 vacation days, Wed holiday bridges)
    - Mon-Tue + Thu-Fri with Wed as working day -> 2 periods
    - Cross-week: Fri + next Mon -> 1 period (weekend bridges)
    - Cross-week: Thu + next Tue with Fri/Mon as working days -> 2 periods
    - Filters to year: dates from 2025 and 2026 mixed, year=2026 -> only 2026 dates grouped
    - Reverse chronological: March period before January period in output
    - Bulgarian singular: 1 day period -> dayCount: 1
  </behavior>
  <implementation>
    Step 1 (RED): Add VacationPeriod interface to lib/vacation/types.ts. Create lib/vacation/periods.test.ts with all test cases above. Create empty lib/vacation/periods.ts exporting a stub `groupVacationPeriods` that returns []. Run tests -- all must FAIL.

    Step 2 (GREEN): Implement groupVacationPeriods in lib/vacation/periods.ts:
    - Import parseISO, addDays, format, getISODay from date-fns
    - Filter vacation dates by year prefix, sort lexicographically
    - Walk sorted dates, check contiguity between consecutive dates
    - isContiguous: walk day-by-day from dateA+1 to dateB-1, check each is weekend (getISODay 6 or 7), holiday (Set lookup), or vacation (Set lookup)
    - makePeriod: create VacationPeriod from days array
    - Reverse final array for most-recent-first order
    - Run tests -- all must PASS

    Step 3 (REFACTOR): Only if obvious cleanup needed. Run tests again to confirm.

    IMPORTANT date-fns rules:
    - Use parseISO() for all date string parsing (Safari safe)
    - Use getISODay() not getDay() for weekday detection (ISO 8601: 1=Mon, 7=Sun)
    - Use format(date, 'yyyy-MM-dd') for date serialization
    - NEVER use new Date(string) directly
  </implementation>
